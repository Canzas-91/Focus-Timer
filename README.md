# Focus Timer

`Focus Timer` — это небольшой веб-проект для концентрации во время работы.  
Приложение помогает чередовать периоды `work` и `break`, запускать фоновую музыку и управлять количеством циклов.

Проект написан на чистом `HTML`, `CSS` и `JavaScript` без использования фреймворков.

## Что умеет проект

- запускать таймер работы
- переключать таймер на перерыв
- проигрывать фоновую музыку
- регулировать громкость
- перематывать трек вперёд и назад
- выбирать количество циклов
- автоматически завершать музыку после последнего цикла
- адаптироваться под разные размеры экрана

## Этапы разработки

### 1. HTML-разметка

Сначала была создана базовая структура страницы в файле `src/Index.html`.

В разметке были добавлены основные блоки:

- область с периодом таймера (`Work / Break`)
- отображение времени
- блок с названием музыки
- иконка громкости и ползунок управления звуком
- кнопки управления таймером и музыкой
- выбор количества циклов
- скрытый тег `audio` для воспроизведения музыки

HTML написан по блочной структуре: каждый смысловой элемент вынесен в отдельный контейнер.  
Это упростило дальнейшую стилизацию и работу с JavaScript.

### 2. CSS-оформление

После создания разметки был написан CSS.

Стили разделены по отдельным файлам:

- `button.css` — кнопки
- `timer.css` — таймер и подписи
- `music.css` — блок музыки и громкости
- `manegment.css` — нижняя панель управления
- `theme.css` — оформление выбора циклов
- `root.css` — переменные, цвета и общие значения

CSS построен модульно: каждый интерфейсный блок оформляется отдельно.  
Это делает код более понятным и позволяет легче изменять отдельные части страницы.

В проекте использованы:

- CSS-переменные
- абсолютное и относительное позиционирование
- `flex` для выравнивания элементов
- скругления, прозрачность и рамки
- медиазапросы для адаптации под планшеты и телефоны

Отдельное внимание было уделено адаптивности.  
Для мобильных экранов были перестроены позиции элементов, чтобы название музыки, период и выбор циклов выглядели аккуратно и не мешали основному таймеру.

### 3. JavaScript-логика

После верстки была добавлена логика приложения на `JavaScript`.

Код разбит по файлам:

- `index.js` — точка входа
- `buttons.js` — обработка кнопок управления
- `workTime.js` — логика рабочего таймера
- `breakTime.js` — логика таймера перерыва
- `manegment.js` — управление громкостью

JavaScript написан через `ES modules`, поэтому логика разделена на независимые части и связывается через `import/export`.

### Как устроен JavaScript

#### `workTime.js`

Здесь находится логика рабочего таймера:

- хранится время работы
- запускается и останавливается интервал
- обновляется отображение таймера
- при окончании рабочего времени запускается перерыв

#### `breakTime.js`

В этом файле находится логика перерыва:

- запускается отдельный таймер перерыва
- после завершения перерыва проверяется количество циклов
- если циклы ещё остались, запускается новый рабочий период
- если цикл последний, музыка останавливается

#### `buttons.js`

Этот файл отвечает за взаимодействие с кнопками:

- запуск и пауза таймера
- запуск и остановка музыки
- перемотка трека вперёд
- перемотка трека назад
- чтение выбранного количества циклов

#### `manegment.js`

Здесь реализовано управление громкостью:

- по нажатию на иконку появляется или скрывается ползунок
- значение ползунка передаётся в `audio.volume`

## Как работает приложение

1. Пользователь выбирает количество циклов.
2. Нажимает кнопку запуска.
3. Начинается период `Work`.
4. Когда рабочее время заканчивается, автоматически начинается `Break`.
5. После завершения перерыва запускается следующий цикл.
6. После последнего цикла музыка останавливается, а сессия завершается.
